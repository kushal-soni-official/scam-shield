About ScamShield

ScamShield is an AI-powered tool designed to detect common Indian scams (UPI fraud, digital arrest, lottery scams, refund tricks, government scheme frauds, etc.) using TF-IDF vectorization and a PyTorch neural network.

Key Features
  • Batch classification
  • Confidence scoring
  • Live dashboard
  • History log
  • PDF export
Technical Details
  • Accuracy on test set: ~95.65%
  • Log file: scam_log.txt
  • Model: TF-IDF + PyTorch
  • Built for educational use
How It Works

Text is transformed into numerical features using TF-IDF.

A PyTorch neural network analyzes patterns and context.

Results are displayed with confidence scores and logged for review.

Impact & Awareness

In 2025 alone, over 1.5 million cyber fraud cases were reported in India. ScamShield helps you stay one step ahead by identifying scam messages before they cause harm. Remember: never share OTPs, bank details, or click suspicious links.

Together, we can build a safer digital India.

Stay vigilant. Report confirmed scams to cyber helpline 1930.
